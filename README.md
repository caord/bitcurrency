# Bitcoin Currency Trader UI

A Qt-based GUI for trading currency and bitcoin through Blockchain

## Requirements

* Python 2.7
* PyQt 4.8
* NumPy
* PyQtGraph
* QDarkStyle

## Installation

1. Download Source
2. Run ./go.py
